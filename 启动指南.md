## 🎯 启动方法

### 方法一：双击启动（推荐）
- **Windows 用户**: 双击 `start.bat` 文件
- **PowerShell 用户**: 右键 `start.ps1` → "使用 PowerShell 运行"

### 方法二：命令行启动
```bash
# 打开命令行/CMD/PowerShell
cd "d:/超级无敌nb的无敌大网站/kobe-portfolio"
npm start
```

### 方法三：开发模式（自动重启）
```bash
npm run dev
```

## 🔧 故障排除

### 端口被占用
修改 `.env` 文件中的 `PORT=3000` 为其他端口号（如 3001）

### 依赖问题
```bash
npm cache clean --force
npm install
```

### 权限问题（PowerShell）
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
