## 🚀 部署指令

### 方法一：快速部署（推荐）
```bash
npm run deploy:windows  # Windows
npm run deploy:unix    # Linux/Mac
```

### 方法二：手动部署
```bash
# 1. 登录 CloudBase
cloudbase login

# 2. 部署云函数
cloudbase functions:deploy

# 3. 部署静态网站
cloudbase hosting deploy index.html style.css script.js Profile.png hero-bg.jpg

# 4. 查看部署状态
cloudbase hosting list
```

### 方法三：自动化部署
```bash
node quick-deploy.js
```
